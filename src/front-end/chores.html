Chores List

Here is a list of all our chores.

You can see assigned chores according to the person in charge for the week.

*Important: Make sure to add at least one roommate in order to use this functionality. Please click on the "Roommate List" tab and add a roommate by following the instruction in the page.

Usage:

a) Adding a chore
  1. Select the week that you want to see by adjusting the week with the right and left arrow button.
  2. Click the + button to add a chore by filling in the chore name & selecting the roommate assigned to the chore this week. You can also add a description for the chore. Click Continue when this section is done.
  3. You should now be in the "Assignees" section. The chore will rotate through one roommate each week, and the "assignees" are the roommates included in the cycle. Select the roommates to include in the cycle and press Create to finish creating the chore.

b) Editing a chore
  1. Click on the chore card that you want to edit.
  2. Edit the information and click Continue when done.
  3. Change assignees if needed. Click the "Edit" button when done. Check the rotation for each week with the arrow buttons next to the current week!
  4. Click on the Delete button at any point if you want to delete this chore.
